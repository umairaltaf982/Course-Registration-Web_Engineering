# 🎓 University Course Registration System

## 📌 Overview
This is a **modern, interactive course registration system** designed for university students and administrators.  
It allows students to **register for courses, view schedules, and check seat availability** in real time.  
Admins can **manage courses, oversee student enrollments, and generate reports** efficiently.

---

## 🚀 Features

### **🎓 Student Features**
✅ **Login with Roll Number** (No self-registration)  
✅ **Interactive Weekly Schedule** (Detects time conflicts dynamically)  
✅ **Real-Time Seat Availability** (No page refresh required)  
✅ **Course Filtering** (Search by department, time, availability, etc.)  
✅ **Persistent Schedule State** (Keeps selected courses during session)  
✅ **Prerequisite Verification** (Shows required courses before registration)  

### **🛠 Admin Features**
✅ **Login with Username & Password**  
✅ **Manage Courses** (Add, update, delete, set prerequisites)  
✅ **Manage Students** (View & override registrations)  
✅ **Seat Management** (Adjust course seats dynamically)  
✅ **Generate Reports** (Registered students, open seats, missing prerequisites)  

---

## 🏗️ Tech Stack
- **Frontend:** HTML, CSS, EJS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** Enforced via roll numbers (students) and passwords (admins)
